## POST - 管理员登录
### 可选参数形式
无

#### 上传数据
| 字段     | 类型   | 示例       | 说明           |
| -------- | ------ | ---------- | -------------- |
| phone    | String | "12312313" | 管理员手机号码 |
| password | String | "FASDFSF"  | 管理员密码     |

#### 返回数据
| 字段    | 类型   | 示例 | 说明   |
| ------- | ------ | ---- | ------ |
| errcode | Number | 0    | 错误码 |
