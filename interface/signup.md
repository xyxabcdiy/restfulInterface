# /signup
## POST - 新建一个用户
### 可选参数形式
无

#### 上传数据
| 字段     | 类型   | 示例          | 说明             |
| -------- | ------ | ------------- | ---------------- |
| phone    | String | "12122342343" | 用户手机号       |
| password | String | "FASDFSF"     | 用户密码         |
| vcode    | Number | 23232         | 用户收到的验证码 |

#### 返回数据
| 字段    | 类型   | 示例 | 说明       |
| ------- | ------ | ---- | ---------- |
| errcode | Number | 0    | 操作错误码 |
